## 🔧 RTCFR Framework Explained

### Role
The persona or expertise the AI should adopt
```
Example: "You are a blockchain educator specializing in crypto for beginners"
```

### Task
The specific action or deliverable requested
```
Example: "Create a 500-word explanation of Beldex for high school students"
```

### Context
Background information and constraints
```
Example: 
- Audience: High school students (age 14-18)
- Tone: Friendly and relatable
- Format: Story-based explanation
- Goal: Generate interest in blockchain
```

### Few-Shot Example
Demonstration of expected input/output quality
```
Example:
Input: "Explain blockchain"
Output: "Blockchain is like a digital record book that everyone can see"
```

### Report
Detailed structure of expected output
```
Example: 
- 5 bullet points maximum
- One real-world analogy
- Simple vocabulary
- Actionable next steps
```

## 🐛 Troubleshooting

### Problem: Output doesn't match expectations
**Solution**: 
- Add more specific examples
- Define format more clearly
- Specify tone/style
- Add constraints

### Problem: AI gives too much/too little content
**Solution**:
- Specify exact word count
- Define number of bullet points
- Set maximum sections
- Provide output examples

### Problem: Response lacks relevant context
**Solution**:
- Add more background in "Context"
- Include real-world examples
- Define audience more specifically
- Add relevant constraints

### Problem: Tone doesn't match brand voice
**Solution**:
- Specify tone clearly ("professional," "casual," etc.)
- Provide tone examples
- Add brand guidelines
- Revise and iterate
