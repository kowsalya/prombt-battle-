## 🎓 Understanding the RTCFR Framework

### **R**ole
Who is the AI speaking/acting as?
```
Example: "You are a blockchain educator specializing in crypto for beginners"
```

### **T**ask
What specific action is being requested?
```
Example: "Create a 500-word explanation of Beldex for high school students"
```

### **C**ontext
Background, audience, goals, tone, format, restrictions
```
Example:
- Audience: High school students (age 14-18)
- Tone: Friendly and relatable
- Format: Story-based explanation
- Goal: Generate interest in blockchain
```

### **F**ew-Shot Examples
Demonstration of expected input/output quality
```
Example:
Input: "Explain blockchain"
Output: "Blockchain is like a digital record book that everyone can see"
```

### **R**eport
Detailed structure of expected output
```
Example: 
- 5 bullet points maximum
- One real-world analogy
- Simple vocabulary
- Actionable next steps
```
